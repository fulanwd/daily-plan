日课小组件 · 两种方案
====================

方案 A · 系统日历小组件（零代码，推荐先试）
-----------------------------------------
你已经订阅了 calendar.ics 的话：
1. 主屏幕长按 → 左上角「+」
2. 搜索「日历」
3. 选「今日日程」或「即将开始」小组件
4. 添加后显示的就是日课里的安排

方案 B · Scriptable 小组件（更像定制日课）
-----------------------------------------
需要 App Store 免费安装「Scriptable」

1. 打开 Scriptable → 右上角「+」新建脚本
2. 把 rike-widget.js 全部内容粘贴进去
3. 脚本命名「日课」，点完成
4. 主屏幕长按 → 「+」→ 搜索 Scriptable
5. 选小组件尺寸 → 脚本选「日课」

小组件会显示：当前在做什么 / 下一步是什么
点击小组件打开完整日课网站

注意：Scriptable 需要联网拉取 plans.json
更新计划后，小组件约 15–30 分钟自动刷新（或编辑脚本点运行强制刷新）

方案 C · 一览页（不是真小组件，但可快速查看）
-----------------------------------------
浏览器打开：https://fulan-daily-plan.netlify.app/widget.html
可添加到主屏幕当快捷方式
